# 第6章 NPML检测器

## 6.1 引言

PRML技术是PR(部分响应)均衡器和维特比检测器的联合使用，广泛应用于硬盘驱动器的信号处理系统中，如第4章所述。考虑如图6.1所示的D域等效离散时间信道模型，其中A(D)是输入比特数据，C(D)是信道，N(D)是加性高斯白噪声(AWGN)，F(D)是PR均衡器，H(D)是目标，Y(D)是送入维特比检测器进行数据译码的数据，而$\hat { A } ( D )$是输入比特数据A(D)的估计值。

![](../source_final/images/chapter_6/96e0673b5e8923fae41b47b6e24ca92e4217605dbbdd0d999db6d6d69d549f47.jpg)  
图6.1: 等效离散时间信道模型

由图6.1，均衡器的输出数据Y(D)可以写成如式(4.6)所示的数学方程，即

$$
Y ( D ) = \underbrace { A ( D ) H ( D ) } _ { \mathrm { w a n t e d ~ s i g n a l } } + \underbrace { N ( D ) { \frac { H ( D ) } { C ( D ) } } } _ { W ( D ) }\tag{6.1}
$$

其中W(D)是进入维特比检测器的噪声(如4.2.2节所述)。一般来说，维特比检测器只有在W(D)具有加性高斯白噪声特性时才能最有效地工作。然而在实际中(特别是当系统在高硬盘驱动器数据密度下工作时)，设计具有少量抽头的目标H(D)使其响应等于信道C(D)是非常困难的。因此，通常W(D)具有有色噪声(colored noise)[25]的特性，这将导致维特比检测器的性能下降。所以，如果要使维特比检测器能够像以前一样高效工作，研究人员必须找到某种方法，在将Y(D)中的噪声分量(即W(D))送入维特比检测器进行译码之前，使其具有加性高斯白噪声的特性。

因此可以说，NPML技术[51,52]是PR均衡器与嵌入了噪声预测过程(即噪声白化过程)的维特比检测器的联合使用，如图6.2所示。因此，当系统在高硬盘驱动器数据密度下工作时，应优先使用NPML检测器而非PRML检测器，以获得更好的系统整体性能。

## 6.2 噪声预测过程

数字信号处理系统中常用的预测滤波器既有有限冲激响应(FIR: finite impulse response)类型，也有无限冲激响应(IIR: infinite impulse response)类型。此外，预测滤波器的一般特性是，预测误差随着预测滤波器抽头数量的增加而逐渐减小。本书仅考虑FIR型预测滤波器。

考虑如图6.3所示的噪声白化过程模型，其中$w _ { k }$是有色噪声，$\hat { w } _ { k }$是w_k的估计值，$e _ { k } = w _ { k } - \hat { w } _ { k }$是预测误差，而$P ( D )$是预测滤波器在$D$域中的传递函数，其方程形式为

$$
P ( D ) = \sum _ { k = 1 } ^ { N } p _ { k } D ^ { k } = p _ { 1 } D + p _ { 2 } D ^ { 2 } + . . . + p _ { N } D ^ { N }\tag{6.2}
$$

![](../source_final/images/chapter_6/11445b13cbe8bb5cb5ce3639360611c85213904a6bdfa24381f54ab48ebecb13.jpg)  
图6.3: 噪声白化过程模型

其中$p _ { k }$是预测滤波器的第k个系数，N是预测滤波器的总抽头数。因此$\hat { w } _ { k }$可以写成数学方程的形式，即

$$
\hat { w } _ { k } = \sum _ { i = 1 } ^ { N } p _ { i } w _ { k - i }\tag{6.3}
$$

式(6.3)被称为"线性一步预测器(linear one-step predictor)"。同样地，预测误差$e _ { k }$可以表示为如下的数学方程

$$
e _ { k } = w _ { k } - { \hat { w } } _ { k } = w _ { k } - \sum _ { i = 1 } ^ { N } p _ { i } w _ { k - i }\tag{6.4}
$$

或者写成$D$域的形式，即

$$
E ( D ) = [ 1 - P ( D ) ] W ( D )\tag{6.5}
$$

其中$[ 1 - P ( D ) ]$项通常被称为"预测误差滤波器(prediction error filter)"。

## 6.3 预测滤波器系数的求解

设计预测滤波器$P ( D )$(即求解预测滤波器的系数)的目的是使预测误差$e _ { k }$最小化，换句话说，就是使$e _ { k }$尽可能具有白噪声的特性。这是因为数据$e _ { k }$被视为送入维特比检测器进行译码的数据中残留的噪声分量。因此，求解预测滤波器系数的最佳方法是使均方误差(MSE: mean-squared error)[51,52]

$$
E \left[ e _ { k } ^ { 2 } \right] = E \left[ ( w _ { k } - \hat { w } _ { k } ) ^ { 2 } \right]\tag{6.6}
$$

最小化，其中$E [ \cdot ]$是期望算子。这可以通过对式(6.6)关于每个预测滤波器系数$p _ { i }$求导，令结果为零，然后求解线性方程组得到，或者利用"正交性原理(orthogonality principle)"，即

$$
E \left[ ( w _ { k } - \hat { w } _ { k } ) w _ { m } \right] = 0\tag{6.7}
$$

对于$m = 1 , 2 , \ldots , N$。求解式(6.7)可得

$$
\begin{array} { r c l } { { E [ w _ { k } w _ { m } ] - \displaystyle \sum _ { i = 1 } ^ { N } p _ { i } E [ w _ { k - i } w _ { m } ] } } & { { = } } & { { 0 } } \\ { { } } & { { } } & { { } } \\ { { E [ w _ { k } w _ { m } ] } } & { { = } } & { { \displaystyle \sum _ { i = 1 } ^ { N } p _ { i } E [ w _ { k - i } w _ { m } ] } } \\ { { } } & { { } } & { { } } \\ { { R _ { w w } ( k - m ) } } & { { = } } & { { \displaystyle \sum _ { i = 1 } ^ { N } p _ { i } R _ { w w } ( k - i - m ) } } \end{array}\tag{6.8}
$$

其中$R _ { w w } ( i )$是噪声$w _ { k }$的第$i$阶自相关(auto-correlation)，定义为

$$
R _ { w w } ( i ) = E [ w _ { k + i } w _ { k } ] = E \left[ \sum _ { k = 0 } ^ { S - 1 } w _ { k + i } w _ { k } \right]\tag{6.9}
$$

其中$S$是序列$\{ w _ { k } \}$的长度或比特数。将$k - m = j$代入式(6.8)，得到"正规方程(normal equation)"，即

$$
R _ { w w } ( j ) = \sum _ { i = 1 } ^ { N } p _ { i } R _ { w w } ( j - i )\tag{6.10}
$$

对于$j = 1 , 2 , \dots , N$，可以写成矩阵形式，即

$$
\underbrace { \left[ \begin{array} { c } { R _ { w w } ( 1 ) } \\ { R _ { w w } ( 2 ) } \\ { \vdots } \\ { R _ { w w } ( N ) } \end{array} \right] } _ { \mathbf { r } } = \underbrace { \left[ \begin{array} { c c c c c c } { R _ { w w } ( 0 ) } & { R _ { w w } ( 1 ) } & { \hdots } & { R _ { w w } ( N - 1 ) } \\ { R _ { w w } ( 1 ) } & { R _ { w w } ( 0 ) } & { \hdots } & { R _ { w w } ( N - 2 ) } \\ { \vdots } & { \vdots } & { \vdots } & { \vdots } \\ { R _ { w w } ( N - 1 ) } & { \hdots } & { R _ { w w } ( 1 ) } & { R _ { w w } ( 0 ) } \end{array} \right] } _ { \mathbf { R } } \underbrace { \left[ \begin{array} { c } { p _ { 1 } } \\ { p _ { 2 } } \\ { \vdots } \\ { p _ { N } } \end{array} \right] } _ { \mathbf { p } }\tag{6.11}
$$

或者

$$
\mathbf { r } = \mathbf { R } \mathbf { p }\tag{6.12}
$$

由于R是方阵，因此预测滤波器的系数P可以通过求解式(6.12)得到，即

$$
\mathbf { p } = \mathbf { R } ^ { - 1 } \mathbf { r }\tag{6.13}
$$

预测滤波器的最小均方误差(MMSE: minimum mean-squared error)等于[52]

$$
E \left[ e _ { k } ^ { 2 } \right] = R _ { w w } ( 0 ) - \sum _ { i = 1 } ^ { N } p _ { i } R _ { w w } ( i )\tag{6.14}
$$

## 6.4 NPML检测器的工作原理

为了便于解释NPML检测器的工作原理，考虑如图6.4所示的PR4信道模型，其中NPML检测器输入侧的数据可以写成数学方程的形式，即

$$
\begin{array} { l c l } { { y _ { k } } } & { { = } } & { { r _ { k } + w _ { k } } } \\ { { } } & { { } } & { { } } \\ { { } } & { { = } } & { { a _ { k } - a _ { k - 2 } + w _ { k } } } \end{array}\tag{6.15}
$$

其中$r _ { k } = a _ { k } - a _ { k - 2 }$是信道输出数据，$w _ { k }$是有色噪声。

![](../source_final/images/chapter_6/36ac46a3a70a351cc1e6e1bcd2133b9cec3c269bba1f9bc5fb92384b998be0e2.jpg)  
图6.4: PR4信道模型

NPML检测器的工作原理与维特比检测器在分支度量的计算上有所不同。即NPML检测器的分支度量包含了噪声预测值$\hat { w } _ { k }$项，即

$$
\lambda _ { k } ( u , q ) = | y _ { k } - \hat { r } _ { k } ( u , q ) - \hat { w } _ { k } | ^ { 2 }\tag{6.16}
$$

其中$\hat { r } _ { k } ( u , q )$是与从状态u到状态q的转移相对应的无噪声信道输出。设$a _ { k } ( q )$是在时刻k与从状态u到状态q的转移路径相对应的输入比特数据，则对于PR4信道有

$$
\hat { r } _ { k } ( u , q ) = a _ { k } ( q ) - a _ { k - 2 } ( q )\tag{6.17}
$$

此外，噪声预测值$\hat { w } _ { k }$可以写成数学方程的形式，即

$$
\hat { w } _ { k } = \sum _ { i = 1 } ^ { N } p _ { i } w _ { k - i }\tag{6.18}
$$

将式(6.15)中的$w _ { k } = y _ { k } - a _ { k } + a _ { k - 2 }$代入式(6.18)，得到

$$
{ \hat { w } } _ { k } = \sum _ { i = 1 } ^ { N } p _ { i } \left( y _ { k - i } - a _ { k - i } + a _ { k - i - 2 } \right)\tag{6.19}
$$

将式(6.17)中的$\hat { r } _ { k } ( u , q )$和式(6.19)中的$\hat { w } _ { k }$代入式(6.16)，得到

$$
\lambda _ { k } ( u , q ) = \left. y _ { k } - a _ { k } ( q ) + a _ { k - 2 } ( q ) - \sum _ { i = 1 } ^ { N } p _ { i } \left( y _ { k - i } - \hat { a } _ { k - i } ( q ) + \hat { a } _ { k - i - 2 } ( q ) \right) \right. ^ { 2 }\tag{6.20}
$$

其中$\hat { a } _ { k } ( \boldsymbol q )$是在时刻k与到达状态q的幸存路径相对应的输入比特数据估计值。

式(6.20)中的分支度量不适合用于需要高处理速度的应用(如硬盘驱动器)，因为分支度量的计算增加了乘法运算(用于噪声预测)，而不像普通维特比算法中仅有加-比-选(ACS: add-compare-select)运算。因此，为了使NPML检测器能够用于需要高处理速度的应用，式(6.20)必须重新整理为

$$
\lambda _ { k } ( u , q ) = \left| z _ { k } - \sum _ { i = K + 1 } ^ { N + 2 } \hat { a } _ { k - i } ( q ) g _ { i } + \sum _ { i = 1 } ^ { K } a _ { k - i } ( q ) g _ { i } - a _ { k } ( q ) \right| ^ { 2 }\tag{6.21}
$$

其中K是在NPML检测器的复杂度和性能之间进行折衷的参数，即K越大，复杂度越高但性能越好(反之亦然)，

$$
z _ { k } = y _ { k } - \sum _ { i = 1 } ^ { N } y _ { k - i } p _ { i }\tag{6.22}
$$

是预测误差滤波器$[ 1 - P ( D ) ]$的输出数据，如图6.5所示，而g_i是D域中"有效目标(effective target)"的系数，定义为

$$
\begin{array} { l c l } { { H _ { \mathrm { e f f } } ( D ) } } & { { = } } & { { 1 - g _ { 1 } D - g _ { 2 } D ^ { 2 } - \ldots - g _ { N + 2 } D ^ { N + 2 } } } \\ { { } } & { { } } & { { } } \\ { { } } & { { = } } & { { ( 1 - D ^ { 2 } ) [ 1 - P ( D ) ] } } \end{array}\tag{6.23}
$$

综上所述，在实际中构建NPML检测器的方法如下：

1) 使用式(6.13)计算预测滤波器P(D)。

2) 通过将对应于所需目标H(D)的均衡器输出序列$\left\{ y _ { k } \right\}$通过预测误差滤波器[1− P(D)]，从式(6.22)计算序列{zk}，如图6.5所示。

![](../source_final/images/chapter_6/4fd03d3c79c2830de93ecd347b272d0d2c7bf3d8eecff2321dbf5d91df6416cb.jpg)  
图6.5: 用于高处理速度应用的NPML检测器结构

3) 将所得序列$\{ z _ { k } \}$送入普通维特比检测器进行数据译码，但维特比算法中使用的网格图(如4.3.3节所述)必须由有效目标$H _ { \mathrm { e f f } } ( D )$构建，其计算公式为

$$
H _ { \mathrm { e f f } } ( D ) = H ( D ) [ 1 - P ( D ) ]\tag{6.24}
$$

其中$H ( D )$是与系统中使用的均衡器相对应的目标。可以看出，有效目标的系数是实数，即尽管$H ( D )$的系数是整数，但$P ( D )$的系数是实数。因此，得到的$H _ { \mathrm { e f f } } ( D )$的系数是实数。因此可以说，有效目标$H _ { \mathrm { e f f } } ( D )$也是一种GPR目标。

## 6.5 实验结果

本节将使用如图6.10所示的水平记录(longitudinal recording)系统的信道模型，对PRML和NPML检测器的性能进行比较。

![](../source_final/images/chapter_6/d704c7a8ab86ad76a7351189ecc082e3af32ab4e18e907af3885a7e335e6d8ae.jpg)  
图6.10: 磁记录系统的信道模型

读回信号可以写成数学方程的形式，即

$$
p ( t ) = \sum _ { k = 0 } ^ { S - 1 } b _ { k } g ( t - k T ) + n ( t )\tag{6.25}
$$

其中$b _ { k } = ( a _ { k } - a _ { k - 1 } ) / 2$是转换比特($b _ { k } = \pm 1$对应于正或负的转换，$b _ { k } = 0$表示无转换)，$a _ { k } \in \pm 1$是总数为$S = 4 0 9 6$比特(即1个扇区)的输入比特，$g ( t )$是按照式(1.1)的转换脉冲信号，$n ( t )$是双边功率谱密度为$N _ { 0 } / 2$的加性高斯白噪声。

读回信号$p ( t )$将通过7阶巴特沃斯低通滤波器，并以采样频率$1 / T$进行采样，假设采样过程在读回信号和采样电路之间具有完美的同步(perfect synchronization)。然后，输出数据序列$\{ s _ { k } \}$被送入均衡器(equalizer)以将信号波形调整到所需目标，再将所得输出数据序列$\left\{ y _ { k } \right\}$送入检测器(detector)进行数据译码，以找到最可能的输入数据序列$\{ a _ { k } \}$的估计值。这里使用的SNR定义为

$$
\mathrm { S N R } = 1 0 \log _ { 1 0 } \left( \frac { { V _ { p } } ^ { 2 } } { \sigma ^ { 2 } } \right)\tag{dB}
$$

(6.26)

其中$V _ { p } = 1$是单个独立转换脉冲(isolated transition pulse)的最大幅度，$\sigma ^ { 2 } = N _ { 0 } / ( 2 T )$是噪声$n ( t )$的功率。此外，每个误比特率(BER)数据点通过使用多个扇区的数据进行计算，直到获得大于或等于1000比特的误差。

![](../source_final/images/chapter_6/32ada0763c9d0bf4e955951d0aaeeb9f410e676fbb19d336bbb1da31e8231588.jpg)  
图6.11: ND = 2时各系统的性能

图6.11比较了ND = 2时PRML和NPML检测器的性能，设所有系统使用PR4目标$H ( D ) = 1 - D ^ { 2 }$。因此，PRML检测器网格图中使用的状态总数为$2 ^ { 2 } = 4$个状态，而NPML检测器网格图中使用的状态总数为$2 ^ { 2 + 2 } = 1 6$个状态(对于2抽头预测滤波器)和$2 ^ { 2 + 4 } = 6 4$个状态(对于4抽头预测滤波器)。可以看出，NPML检测器比PRML检测器更复杂，但从图6.11的实验结果可以发现，NPML检测器的性能明显优于PRML检测器。或者说，在$\mathrm { B E R } = 1 0 ^ { - 4 }$水平上，NPML检测器的性能比PRML检测器好约2 dB。此外，从实验结果还发现，使用2抽头预测滤波器的NPML检测器与使用4抽头预测滤波器的性能接近。因此，对于所考虑的系统，使用2抽头预测滤波器的NPML检测器已经足够，因为其性能与4抽头预测滤波器相近，但复杂度要低得多。

## 6.6 本章小结

当系统在高硬盘驱动器数据密度下工作时，维特比检测器输入侧的噪声分量具有更强的有色噪声(colored noise)特性。在这种情况下，PRML检测器无法高效工作。因此，NPML检测器被用于提高系统性能，这是因为NPML检测器将预测滤波器(用于将有色的噪声转换为白色噪声)与维特比检测器结合使用来进行数据译码。此外，还发现用于构建NPML检测器网格图的有效目标可以被视为GPR目标，这也是NPML检测器性能优于PRML检测器的原因之一，因为如第3章所述，GPR目标的性能优于PR目标。

尽管NPML检测器的性能优于PRML检测器，但NPML检测器更复杂。因此，在决定是否使用NPML检测器时，应考虑所获得的性能提升是否值得随之而来的复杂度增加。
